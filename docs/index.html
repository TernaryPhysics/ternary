Features

Kernel-Level Routing

XDP/TC programs for line-rate packet processing. No userspace overhead.

ML Health Detection

Ternary neural network predicts backend health from TCP metrics in real-time.

Automatic Failover

Routes traffic away from degraded backends before they fail completely.

Full Observability

Prometheus metrics with per-backend health scores and routing statistics.

How It Works

BPF programs run in the kernel, processing packets at line rate. A ternary neural network evaluates backend health from TCP metrics, automatically routing traffic away from degraded servers.

Quick Start

Requirements

  • Kernel Linux 5.8+
  • Tools bpftool, iproute2
  • Arch x86_64, aarch64

Install

# One-line install
curl -sSL https://github.com/TernaryPhysics/bpf-loadbalancer/releases/latest/download/install.sh | sudo bash
# Attach to interface
sudo bpf-loadbalancer attach eth0
# Add backends
sudo bpf-loadbalancer add-backend 10.0.0.10:8080
sudo bpf-loadbalancer add-backend 10.0.0.11:8080
