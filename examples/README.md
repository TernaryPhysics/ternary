# TernaryPhysics Examples

Demos you can run without kernel access to understand how TernaryPhysics works.

## Quick Start

```bash
cd examples
pip install -r requirements.txt
python demo.py
```

## What's Included

### demo.py
Interactive demo showing:
1. Ternary neural network basics (weights of -1, 0, +1)
2. Packet feature extraction
3. Inference (classification)
4. Feedback loop (learning from outcomes)

### simulate_traffic.py
Generates synthetic traffic patterns and shows how the model learns to classify them over time.

## Requirements

- Python 3.10+
- NumPy

No kernel access, root, or special hardware required.
